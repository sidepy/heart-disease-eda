# ❤️ Heart Disease Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project explores the UCI Heart Disease dataset to identify the most informative features associated with heart disease.

## 📊 Objectives
- Analyze demographic and clinical variables
- Identify strong predictors of heart disease
- Examine feature relationships using correlation analysis
- Provide summary insights and limitations

## 🔍 Key Findings
- Strong predictors: `ca`, `thal`, `slope`, `exang`
- Moderate predictors: `oldpeak`, `thalach`
- Weak predictors: `chol`, `fbs`
- Clinical diagnostic variables were more informative than general health indicators

## 🛠 Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Jupyter Notebook

## 📁 Dataset
UCI Heart Disease Dataset

This project demonstrates data cleaning, visualization, statistical interpretation, and structured exploratory data analysis.
